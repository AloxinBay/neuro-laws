# 🧠 Нейро-памятка по законам GTA 5 RP (San-Andreas)

Своя нейросеть-ассистент по законам серверов **Memphis** и **Orlando**. Дообучается
на текстах законов и отвечает на вопросы кратко и осмысленно.

## Что умеет
- Отвечает по номеру статьи: спрашиваешь `12.8` — получаешь краткую выжимку по частям с наказаниями.
- Понимает уровень розыска **★** (от ★ до ★★★★★).
- Краткий формат вместо простыни текста:
  ```
  Ст. 12.8 ★★★★ — Незаконный оборот оружия или спец. средств
  ч1 — незаконное приобретение, передача, сбыт, перевозка оружия (4 года л/с или штраф $10k до $30k)
  ч2 — незаконное хранение, ношение или использование оружия (4 года л/с или штраф $10k до $30k)
  ```
- Понимает **русский и английский** (основной — русский, отвечает на языке вопроса).
- Подробный режим: `подробно 12.8`.
- Поиск по смыслу: `какая статья за убийство`.

## Структура проекта
```
laws/                     # исходные тексты законов (Memphis, Orlando)
scripts/
  parse_laws.py           # laws/*.txt  ->  data/laws.json (структура)
  build_dataset.py        # data/laws.json  ->  data/train.jsonl (вопрос-ответ)
  inference.py            # локальный чат с обученной моделью
notebooks/
  train_colab.ipynb       # обучение на Google Colab (на полных ресурсах GPU)
data/                     # сгенерированные laws.json и train.jsonl
requirements.txt
```

## Как обучить на Google Colab
1. Залей этот проект на GitHub под аккаунтом **AloxinBay** (например, репозиторий `neuro-laws`).
2. Открой `notebooks/train_colab.ipynb` в Google Colab.
3. `Среда выполнения → Сменить среду выполнения → GPU` (A100/L4 быстрее, T4 тоже подойдёт).
4. В ячейке №2 укажи `REPO_NAME` (имя своего репозитория).
5. `Среда выполнения → Выполнить все`.

Ноутбук сам: склонирует репозиторий → поставит зависимости → распарсит законы →
соберёт датасет (русский + английский) → дообучит модель (QLoRA на Qwen2.5) →
сольёт адаптер с моделью → проверит на тестовых вопросах.

Готовую модель можно сохранить на Google Drive или Hugging Face (ячейка №11).

## Локальная подготовка данных (без обучения)
```bash
pip install -r requirements.txt
python scripts/parse_laws.py --laws-dir laws --out data/laws.json
python scripts/build_dataset.py --in data/laws.json --out data/train.jsonl
# с английскими парами (нужен интернет):
python scripts/build_dataset.py --translate
```

## Локальный чат с готовой моделью
```bash
python scripts/inference.py --model outputs/merged
# или базовая модель + адаптер:
python scripts/inference.py --model Qwen/Qwen2.5-3B-Instruct --adapter outputs/lora
```

## Как добавить/обновить законы
Просто положи или отредактируй `.txt` файлы в `laws/<сервер>/` и заново запусти
`parse_laws.py` + `build_dataset.py`, затем переобучи в Colab.

## Модель
По умолчанию `Qwen/Qwen2.5-3B-Instruct` — мультиязычная, хорошо знает русский.
На мощной GPU (A100) можно поставить `Qwen/Qwen2.5-7B-Instruct` в ячейке конфигурации.
