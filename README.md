# 🌙 MoonNeuro — нейро-памятка по законам GTA 5 RP (San-Andreas)

Лёгкий нейро-ассистент по законам серверов **Memphis** и **Orlando**. Ведёт диалог,
отвечает кратко, понимает уровень розыска (★) и сленг игроков. Работает локально и
оффлайн, модель весит **~0.4 ГБ** (формат GGUF Q4).

## Два режима использования

| Режим | Что это | Вес | Болтает |
|-------|---------|-----|---------|
| **MoonNeuro Lite** (`scripts/bot.py`) | поиск по законам без ИИ | ~0 | нет, только поиск |
| **MoonNeuro** (нейросеть) | дообученная Qwen2.5-0.5B в GGUF | ~0.4 ГБ | да |

### Что умеет
```
12.8            → кратко по статье (части + наказания + ★)
12.8 ч1         → конкретная часть
подробно 12.8   → полный текст
убийство        → поиск по словам
кто такой госсник → сленг (госсник=полицейский, крайм=бандит, тулиться=стрелять)
привет / спасибо  → живой диалог
```

Пример краткого ответа:
```
Ст. 12.8 ★★★★ — Незаконный оборот оружия или спец. средств
ч1 — незаконное приобретение, передача, сбыт, перевозка оружия (4 года л/с или штраф $10k до $30k)
ч2 — незаконное хранение, ношение или использование оружия (4 года л/с или штраф $10k до $30k)
```

## Структура
```
laws/                 исходные тексты законов
scripts/
  parse_laws.py       laws/*.txt -> data/laws.json
  build_dataset.py    data/laws.json -> data/train.jsonl (вопрос-ответ + диалоги + сленг)
  bot.py              лёгкий поиск по законам БЕЗ нейросети
  inference.py        чат с моделью через transformers
notebooks/
  train_colab.ipynb   обучение 0.5B + экспорт в GGUF
Modelfile             конфиг для запуска модели в Ollama
MoonNeuro.bat         запуск через Ollama
```

## Вариант 1 — мгновенный поиск без ИИ (рекомендуется для начала)
Ничего обучать не нужно, работает сразу:
```bash
python scripts/parse_laws.py --laws-dir laws --out data/laws.json
python scripts/bot.py
```

## Вариант 2 — нейросеть MoonNeuro (диалог)

### Обучение в Google Colab
1. Открой ноутбук:
   `https://colab.research.google.com/github/AloxinBay/neuro-laws/blob/main/notebooks/train_colab.ipynb`
2. `Среда выполнения → Сменить среду выполнения → GPU`.
3. `Среда выполнения → Выполнить все`.

Ноутбук сам: склонирует репозиторий → соберёт датасет → дообучит Qwen2.5-0.5B →
сожмёт в **GGUF Q4 (~0.4 ГБ)** → даст скачать файл `moonneuro-q4_k_m.gguf`.

### Запуск локально через Ollama (оффлайн)
1. Установи Ollama: https://ollama.com/download
2. Положи скачанный `moonneuro-q4_k_m.gguf` рядом с `Modelfile`.
3. Запусти `MoonNeuro.bat` (или вручную):
   ```bash
   ollama create moonneuro -f Modelfile
   ollama run moonneuro
   ```

## Как добавить законы или сленг
- Законы: положи `.txt` в `laws/<сервер>/`, перезапусти `parse_laws.py` + `build_dataset.py`.
- Сленг: в `scripts/build_dataset.py` добавь строку в словарь `SLANG`.
- Диалоги: там же дополни список `CHAT`.
Затем переобучи модель.
